# 설명
여러 개의 *피커 뷰*와 *이미지 뷰*, *라벨* 등을 이용해서 만든 사진 고르기 앱

# 주요 기능
1. 상단에 **두 개의 피커 뷰**가 있고, 왼쪽 오른쪽 **둘 다** 이미지를 선택할 수 있다.
2. **왼쪽**에서 이미지를 선택한 경우, 가운데에 있는 라벨을 통해 **선택한 이미지의 파일명**을 나타낸다.
3. **오른쪽**에서 이미지를 선택한 경우, 하단에 있는 이미지 뷰를 통해 **선택한 이미지**를 더 크게 나타낸다.
